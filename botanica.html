Taxonomy

Forms found
in the algorithm.

01

Branching angle

Angulus ramificationis

The divergence between parent and child branches — tighter for pines, wider for oaks. Everything else follows from this single number.

02

Attenuation ratio

Ratio attenuationis

How much each successive branch shrinks. Real trees converge toward 0.7 — the ratio that maximises canopy coverage without self-shading.

03

Recursion depth

Profunditas recursiva

How many generations of branches to draw. Nature doesn't have a fixed limit — we cap at 11 to keep it visible to the human eye.

04

Leaf density

Densitas foliorum

The probability that any terminal branch sprouts leaves. Autumn trees get a decay function applied — probability falls as branches tip.

05

Tropism vector

Vector tropismus

The invisible pull — upward for most trees, outward for willows. A subtle bias added to each branch angle, accumulating into the tree's characteristic lean.

06

Stochastic noise

Variatio stochastica

The random seed that makes every tree unique. Same species, same rules, different number — and the whole form shifts. Two oak trees, never alike.
